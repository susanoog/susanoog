# DPDK

- アイディア
	- Kernelのボトルネックを解消
	- ボトルネックの無いアーキテクチャを再設計
- アプローチ
	- CPU専有
	- ユーザランドドライバによりZerocopy
	- Pagefaultを起こさないためのHugepages
- 高性能通信に必要なライブラリの集合
	- Thread Manager
	- Userland Driver
	- Lock-Free Ring
	- Algorithm
	- Memory Manager
